## What is this?

A Progressive Web App that recommends IBM SkillsBuild courses to University of Bristol students based on their academic background, career aspirations, and existing skills. Built end-to-end as part of the IBM SkillsBuild project brief.

## Features

- 🔐 **Firebase Authentication** — email/password sign-up with password recovery
- 🎓 **verified IBM SkillsBuild courses**
- 🎯 **Weighted recommendation engine** with transparent "Why this?" explanations
- 🏫 **Bristol-specific** — 140+ undergraduate and 90+ postgraduate programmes built in
- 📊 **Personal learning dashboard** with real-time progress tracking
- 💬 **Community course reviews** with year + programme attribution
- 📱 **Fully responsive** — installable as a PWA on iOS, Android, Windows, macOS
- ☁️ **Cloud-persistent** — Firestore-backed profile that survives device changes
